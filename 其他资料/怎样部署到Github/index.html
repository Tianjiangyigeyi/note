    怎样部署到github

    怎样部署到Github

    从头开始

    1. 新建文件夹

    2. 用vscode打开新文件夹

    3. 在vscode里新建终端

    4. 依次输入：

    py -m venv venv venv\Scripts\activate

    设置虚拟环境并且激活虚拟环境(目的是与系统环境分开)

    1. pip install mkdocs-material

    (只有第一次要做)

    1. mkdocs new mysite

      (这里的mysite可以是任何名字)

    2. cd mysite

    3. mkdocs serve

    (这时候已经打开了一个网页，随着你vscode里的改动，网页会实时刷新改变)

    1. 接下来则是根据Mkdocs使用说明简单地在mkdocs.yml中简单地设置即可，这时，一个新的项目就建好了

    维护网页

    1. 使用typora做好笔记

    2. 将笔记复制到root文件夹下对应的目录中，编号

    3. 在mkdocs.yml中进行设置，格式如下：

    nav: - 主页: - index.md - 概率论与数理统计: - 第一章: 概率论与数理统计/20230202.md - 第二章: ...

    1. 输入代码如下：

    git init git add . git commit -m "update"//可以不叫update git remote add origin https://github.com/Tianjiangyigeyi/note.git git push -u origin master

    1. 进入mysite文件夹

    cd mysite

    1. 输入代码如下：

    mkdocs gh-deploy

    1. 等待30s左右，网页已成功更新.https://tianjiangyigeyi.github.io/note/
